# kong-admin

[Sails](http://sailsjs.org) application

## install node

```
- install npm
- install n
  `npm install -g n`

- install node 5.6.0
  `n 5.6.0`

```

## publish

- git clone http://xxx/kong-admin.git
- cd kong-admin
- npm install -g sails
- npm install -g nodemon
- npm install
- nodemon -V app # development

```

`production`

```
- npm install pm2 -g
- pm2 start app.js -x -- --prod

```
